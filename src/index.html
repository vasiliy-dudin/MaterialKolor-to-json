Генератор цветовых схем Material Design 3

Создавайте цветовые палитры на основе URL из MaterialKolor.com

Результат

Готов к работе
Вставьте URL из MaterialKolor.com и нажмите "Генерировать JSON"

Информация

Поддерживаемые стили:
  • MONOCHROME - Монохромная схема
  • NEUTRAL - Нейтральная схема
  • TONAL_SPOT - Тональная схема
  • VIBRANT - Яркая схема
  • EXPRESSIVE - Выразительная схема
  • FIDELITY - Точная схема
  • CONTENT - Контентная схема
  • RAINBOW - Радужная схема
  • FRUIT_SALAD - Фруктовая схема
Пример URL:
https://materialkolor.com/?color_seed=FF74BBFF&color_error=FFFFAC29&style=Expressive&color_spec=SPEC_2025
